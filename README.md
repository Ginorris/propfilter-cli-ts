# Prop Filter CLI

A CLI tool for filtering real estate properties.

## Setup

1. Clone the repository:
   ```sh
   git clone https://github.com/Ginorris/propfilter-cli-ts.git
   cd prop-filter-cli
   ```

2. Install Dependencies:
    ```sh
    npm install
    ```

3. Run the CLI:
    ```sh
    npm run dev
    ```
